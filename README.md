# Tailoring Management System

## Overview

This project started from a simple idea while I was working as a tailor during high school.

I wanted to use technology to simplify tailoring work, especially clothing cutting, and make it more accessible for everyone, including people with little tailoring experience.

The project started as a simple database idea and later developed into a web application. Since I did not have a computer, I built and tested the project using only my mobile phone while learning programming step by step.

Although the project is still under development and many planned features have not been completed yet, it has been the most important project in my learning journey. Working on it strengthened my interest in Information Technology and taught me how to solve problems through continuous learning and experimentation.
